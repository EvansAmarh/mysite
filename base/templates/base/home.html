My First Job as a Developer

My journey as a developer began with the RydeMe mobile app project in April 2024, which marked my first hands-on experience in a real-world development setting. As a front-end developer, I was responsible for designing and implementing a responsive and user-friendly interface using React Native and JavaScript. The app provided users with access to rides, food delivery, and item transport services—all from a single platform.

This role challenged me to go beyond theoretical knowledge, pushing me to write efficient code, troubleshoot UI bugs, and ensure compatibility across various mobile devices. It was during this project that I learned the value of cross-functional collaboration, as I worked closely with the back-end team and product designers to integrate real-time features that prioritized user safety, affordability, and fast service delivery. This experience laid a strong foundation for my development career and introduced me to industry standards in front-end development.

Through the RydeMe project, I also developed a stronger appreciation for user experience principles, recognizing how design choices directly impact user satisfaction and retention. I conducted informal usability testing and incorporated user feedback into interface improvements, ensuring the app remained intuitive even as new features were added. This iterative design process taught me to balance aesthetics with functionality, and it sparked a deeper interest in building applications that are not only technically sound but also empathetic to the user's needs and behavior.
